# AO3 Continue

AO3 Continue 是一个面向手机浏览器的 AO3 油猴脚本，用来自动记录每篇作品读到的章节，并在作品页继续到下一章。

## 当前版本

v2.4.0

## 功能

- 自动记录每部作品读到的最高章节
- 回头重看旧章节时不会倒退进度
- 打开作品页/较早章节时提示继续到下一章
- 搜索页、标签页、书签页显示已读进度
- 右下角单浮窗
- 单击浮窗：显示小进度面板
- 长按浮窗：显示导入/导出菜单
- 浮窗可拖动，并记住位置
- 支持 JSON 导入/导出阅读记录

## 安装

打开 Raw 链接安装：

```text
https://raw.githubusercontent.com/Demandedero/AO3-Continue/main/ao3-continue.user.js
```

## 更新

脚本内置 `@updateURL` 和 `@downloadURL`。仓库里的 `ao3-continue.user.js` 更新后，可以在油猴/Tampermonkey 里检查更新。
